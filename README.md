> An AngularJS web client for [The Shelf](http://github.com/groupbuddies/theshelf/)

## Getting Started

Install requirements
```
npm install -g gulp
npm install -g bower
```

Install dependencies
```
bower install
npm install
```

Launch development server
```
# The browser will automatically be opened
gulp dev
```

Run test suite
```
gulp test
```
